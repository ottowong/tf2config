# Otto's TF2 Configs

put these files in this folder:
>\Team Fortress 2\tf\cfg

## General Features
- Set viewmodel FOV to 90
- Use minimal viewmodels

## Class Features
### All Classes
- F1 - Item preset A
- F2 - Item preset B
- F3 - Item preset C
- F4 - Item preset D
- holding A&D or W&S moves you in the last direction you pressed, instead of stopping you
- j - "need a dispenser here"
- h - "thanks"
- f - "spy"
- k - suicide
- autoreload is switched on (overridden on spy)
